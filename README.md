# Грайбер текста песен

Инструмент для автоматической транскрипции аудио в формат LRC (тайм-синкронизированные тексты). Использует модели OpenAI Whisper и WhisperX для распознавания речи с временными метками, поддерживает выделение вокала из аудиомикса и работает через веб-интерфейс или командную строку.

---

## Возможности

- **Два движка транскрипции**: Whisper (сегментные метки) и WhisperX (пословные метки через CTC-выравнивание)
- **Выделение вокала**: Опциональное отделение голоса от инструментала через модели UVR5 перед транскрипцией
- **GPU-ускорение**: Поддержка CUDA (NVIDIA), MPS (Apple Silicon) и CPU-режима с автоматическим определением устройства
- **Многоязычность**: Автоопределение языка или явное указание (русский, английский, украинский, немецкий, французский, испанский, японский, китайский, корейский, турецкий, арабский)
- **Веб-интерфейс**: Тёмная тема, drag-and-drop загрузка файлов до 500 МБ, прямая трансляция прогресса
- **Подстройка тайминга**: Смещение временны́х меток на произвольное количество миллисекунд без повторной транскрипции
- **LRC-экспорт**: Копирование в буфер обмена и скачивание готового `.lrc` файла

## Установка

### Портативный установщик 

1. Скачайте.
2. Запустите `\install.bat` двойным кликом.
3. Установщик автоматически:
   - Загрузит Python 3.12 embeddable (не нужен системный Python)
   - Определит версию CUDA через `nvidia-smi` или предложит выбрать вручную
   - Установит все зависимости (PyTorch, Whisper, WhisperX, Flask и др.)
   - Запустит загрузчик моделей (`download_models.py`) — выберите нужные размеры
   - Создаст готовые ярлыки запуска `run.bat` и `run_web.bat`

> Установка занимает от 5 до 30 минут в зависимости от скорости интернета.

## Запуск

### Веб-интерфейс

```bat
run_web.bat
```

Откройте браузер: **http://localhost:5000**

### Командная строка

```bat
run.bat "путь\к\аудио.mp3"
run.bat "аудио.mp3" -m large-v3 -l ru -o "выход.lrc"
run.bat "аудио.mp3" --print
```

| Аргумент | Описание | По умолчанию |
|---|---|---|
| `-m / --model` | Размер модели Whisper | `medium` |
| `-l / --language` | Код языка (`ru`, `en`, `uk`, `de`…) | авто |
| `-o / --output` | Путь к выходному `.lrc` | рядом с аудио |
| `--print` | Вывести LRC в консоль вместо файла | — |

## Загрузка моделей

Запустите вручную при необходимости:

```bat
venv\Scripts\python.exe installer\download_models.py
```

Или в автоматическом режиме (скачивает модель `medium` + выравниватели `ru` и `en`):

```bat
venv\Scripts\python.exe installer\download_models.py --auto
```

### Доступные модели Whisper

| Модель | Размер | Качество |
|---|---|---|
| tiny | ~75 МБ | минимальное |
| base | ~145 МБ | базовое |
| small | ~465 МБ | среднее |
| **medium** | ~1.5 ГБ | **рекомендуется** |
| large-v2 | ~3 ГБ | высокое |
| large-v3 | ~3 ГБ | максимальное |

### Модели выделения вокала (UVR5)

| Модель | Применение |
|---|---|
| UVR-MDX-NET-Inst_HQ_3 | Универсальная |
| UVR-MDX-NET-Voc_FT | Оптимизирована под вокал |
| Kim_Vocal_2 | Популярная альтернатива |

## Описание функций

### `app.py` — веб-приложение

| Функция | Описание |
|---|---|
| `run_whisper(job_id, audio_path, model_size, language)` | Транскрипция через стандартный Whisper, формирует LRC из сегментов |
| `run_whisperx(job_id, audio_path, model_size, language)` | Транскрипция через WhisperX с пословным выравниванием через CTC-модели |
| `separate_vocals(job_id, audio_path, model_name)` | Выделение вокала через audio-separator (UVR5), возвращает путь к `Vocals.wav` |
| `POST /upload` | Принимает аудиофайл и параметры, ставит задачу в очередь, возвращает `job_id` |
| `GET /progress/<job_id>` | Стриминг прогресса через Server-Sent Events (SSE) |
| `GET /status/<job_id>` | Polling-fallback для статуса задачи (JSON) |
| `GET /download/<job_id>` | Скачивание готового `.lrc` файла |

### `transcribe.py` — CLI

| Функция | Описание |
|---|---|
| `transcribe(audio_path, model_size, language, output_path, print_only)` | Загружает модель Whisper, транскрибирует аудио, сохраняет или выводит LRC |
| `format_lrc(segments)` | Конвертирует сегменты Whisper в строки формата `[MM:SS.cs] текст` |

### `installer/download_models.py` — загрузчик моделей

| Функция | Описание |
|---|---|
| `detect_device()` | Определяет CUDA / MPS / CPU |
| `download_whisper_model(size)` | Загружает модель Whisper заданного размера |
| `download_alignment_model(lang)` | Загружает WhisperX CTC-выравниватель для языка |
| `download_uvr_model(name)` | Загружает модель UVR5 для выделения вокала |
| `interactive_menu()` | Интерактивный выбор моделей через консоль |
| `auto_mode()` | Тихая установка дефолтного набора моделей (флаг `--auto`) |

## Формат вывода LRC

```
[00:03.20] Первая строка текста
[00:07.50] Вторая строка
[00:12.10] Третья строка
```

Формат временно́й метки: `[MM:SS.cs]` (сотые доли секунды).
